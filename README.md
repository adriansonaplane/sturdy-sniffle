# sturdy-sniffle
A Generator WorkShop


## Development runtime

Use Node.js 24.18.0 for local development and CI. The repository includes a `.nvmrc`, so nvm users can run `nvm install && nvm use` from the project root before `npm ci`. Use the npm version bundled with Node.js 24.18.0; do not install or pin a separate npm version for this project.
